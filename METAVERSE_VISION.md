### Cross-Session Contribution

**Between sessions (via checkpoints):**
```bash
# Before session ends
./blackroad-session-persist.sh checkpoint

# Checkpoint saves:
# - Current work in progress
# - Tasks claimed
# - Ideas and plans
# - Contribution state

# New session (could be days/weeks later)
export MY_AGENT="cecilia-coordinator-abc123"
./blackroad-session-persist.sh resume

# Pick up EXACTLY where you left off
# Continue building the metaverse
# Your identity and work persist!
```

1. **Clone the repo** (any time, any agent)

3. **Commit your work** (permanent record)
